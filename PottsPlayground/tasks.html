Graph Coloring¶

class PottsPlayground.Tasks.GraphColoring(ncolors, p=None, nnodes=None, G=None, seed=None)¶

Models the Graph Node coloring problem as a Potts model. Just about the simplest Potts mode design. Rather than asking how many colors are needed to color a graph, which is the typical framing of graph coloring, the simple Potts model form asks if a coloring with a set number of colors is possible.

DisplayState(state)¶

graphically represents the solution given by the supplied state.
